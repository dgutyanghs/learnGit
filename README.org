** 单独恢复一个文件到指定的版本

首先查看该文件的历史版本信息：git log Default@2x.png

恢复该文件：git reset 9aa51d89799716aa68cff3f30c26f8815408e926 Default@2x.png

提交git:git commit -m "revert old file"
